# Upload Folders

Root upload folder for backend files.

- `categories/`: anh danh muc
- `products/`: anh san pham
- `banners/`: banner homepage/landing
- `promotions/`: media khuyen mai
- `blogs/`: anh bai viet
- `pages/`: media trang noi dung tinh
- `stores/`: anh cua hang
- `users/`: avatar nguoi dung
- `brands/`: logo thuong hieu
- `temp/`: file tam trong qua trinh upload/crop

Command to re-create folders:

```bash
php artisan uploads:init
```
